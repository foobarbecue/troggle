About this website

The Erebus Cave and Fumarole Database at http://www.erebuscaves.com was created using free (gratis) and open-source (libre) software. You can download the source for the website using the mercurial command "hg clone -r erebus https://troggle.googlecode.com/hg/" . The backend is in Python, using the Django Web Framework and a MySQL database. The Django project used is the "erebus" branch of Troggle. Maps are all javascript, using the OpenLayers mapping framework. GIMP was used for image editing.

Imagery on the main page is the one exception; the google earth satellite base layer is used for convenience.

The is maintained by Aaron Curtis, aarongc at nmt d0t edu.
